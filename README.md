# Movie Helper


![A screenshot of Movie Helper](/Assets/Images/Screenshot.png)

## About

A website where you type in a movie, and it retrieves the films poster, a trailer, and a short synopsis.

## Technologies Used

* HTML
* CSS / Bootstrap 4
* Javascript / Jquery 

## API used:
